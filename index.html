The strat package: tools for social stratification research

The aim of the R strat package is to bundle various conversion tools, algorithms and visualisation code, useful for social stratification research.

The current (early) development focus is to provide an unified interface to convert between different occupational unit group classifications (e.g. ISCO88) and stratification schema's (e.g. EGP).

Only a part of the code has been validated against existing conversion tools, please do not yet use strat for crucial things!

Current features

  • Functions for converting ISCO88 to EGP (simple), ESeC (simple, full), ISEI, SIOPS, ICAM, Oesch, Güveli.
  • Basic testing/validation framework (using testthat) and labeling-functions.

Tentative roadmap

  • Proper validation and documentation.
  • Extending with additional schema's (ISCO08, Wright, Esping-Anderson).
  • Menu-driven SPSS interface using the SPSS Statistics-R Integration Plug-In.
  • Intergrating R-code for stratification analysis (mobility tables, conditional reference models, etc.).
